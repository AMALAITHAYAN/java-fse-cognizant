Exercise 3: Stored Procedures - PL/SQL Solutions

This folder contains solutions for all 3 scenarios:

1. scenario1_process_monthly_interest.sql
   - Creates stored procedure ProcessMonthlyInterest
   - Applies 1% monthly interest to all SAVINGS accounts

2. scenario2_update_employee_bonus.sql
   - Creates stored procedure UpdateEmployeeBonus
   - Adds bonus percentage to employees in a given department

3. scenario3_transfer_funds.sql
   - Creates stored procedure TransferFunds
   - Transfers money from one account to another after checking sufficient balance

4. exercise3_all_scenarios.sql
   - Combined file containing all 3 stored procedures

5. sample_schema_optional.sql
   - Optional sample tables and data for testing

How to run in Oracle SQL Developer / SQL*Plus:
1. Run sample_schema_optional.sql first if you do not already have tables.
2. Run each scenario file or run exercise3_all_scenarios.sql.
3. Run the test EXEC commands at the bottom of each file if needed.

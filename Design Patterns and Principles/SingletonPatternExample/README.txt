Project Name: SingletonPatternExample

This project demonstrates the Singleton Design Pattern using a Logger class.
The Logger class allows only one object to be created and reused throughout the application.

How to run:
1. Open the project folder in Eclipse, IntelliJ, or VS Code.
2. Open src/LoggerTest.java.
3. Run LoggerTest.

Expected Output:
[LOG] Application started
[LOG] User login checked
Logger 1 hash code: <same value>
Logger 2 hash code: <same value>
Both logger objects are the same instance.
Singleton Pattern implemented successfully.
